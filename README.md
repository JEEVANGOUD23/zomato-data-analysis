# 🍽️ Zomato Data Analysis Using Python

## 📌 Project Overview
This project analyzes Zomato restaurant data to discover customer preferences, restaurant trends, online ordering patterns, ratings distribution, and cost insights.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Analysis Performed
- Data Cleaning and Preprocessing
- Restaurant Type Analysis
- Online Order Analysis
- Ratings Distribution
- Votes Analysis
- Cost for Two Analysis
- Correlation Heatmap

## 📈 Key Insights
- Identified popular restaurant categories.
- Analyzed the impact of online ordering.
- Studied customer ratings and voting patterns.
- Explored spending trends across restaurants.

- Zomato_Data_Analysis_Using_Python.ipynb
- Zomato-data.csv
